CSS Image Hover Effects

Ever wonder how websites add those special little effects on their images? It’s ok, I did too, so I did some digging and found out how some effects are done through CSS. The first and most important step is creating the normal CSS section for your elements, and then a subsequent :hover version of the element. This will actually establish the two stages that the element will transition between when hovering on or removing the cursor from the area.

Once both of those are established, you can begin making your transition stages. The default is the normal section, the one without :hover. This will be what CSS defaults to when you aren’t hovering over it. For a basic hover effect, here is an image that has an overlaid red fill with a little message. This was done by using the hover effect on a div that is in the same parent object of the image, but lower in the hierarchy.

Although this isn’t really an IMAGE hover effect. CSS has ways of applying effects directly on the image itself through the figure element, this is done using CSS transitions! CSS has a plethora of ways to alter images to fit your needs. CSS can allow you to zoom, transform, apply filters, and apply special effects.

The first example here shows zooming in and out. This is done using transform to set scale and transition to set a more specialized ease-in-out transition. Interestingly enough, this is done to the figure element the image is attached to get the desired zoom.

Zoom In & Out

A similar method is used for the rotational effect as seen below. Here, we are using transform to set not only the scale, but also the degrees of rotation that the image is supposed to be at with and without hovering. Fun fact, you can also modify margins through normal transitions to get sliding effects!

Rotate

Next we have filters, which is my personal favorite. This utilizes filter and can have many different keywords used to alter an image. Grayscale() can be used with a percentage argument to make images more or less black-and-white. sepia() with a percentage argument can make an image look old-timey and yellow. You can invert() colorschemes or even blur() them. hue-rotate() takes degree arguments and produces interesting color palettes.

That brings us to special effects! Here, we get to use the animation and keyframe functions of CSS to create little hover animations for our images. The first example is a small flash effect, which animation already has a built in keyword for! It takes a little set-up but once that’s done all you need to do is test out the time arguments! The same is true for the shine animation, but the prep-work for this is a lot more involved. This uses a gradient, skewing, and the ::before to create the shimmer along the image.

Flash

Shine

These hover effects can also be stacked on top of each other. I already teased this back with the rotate images but you can zoom and filter, filter and rotate, even give special effects and filter! With such a wide array of possibilities, you should have no problem getting the right effect for you!
